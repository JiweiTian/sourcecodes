Step of the running project:

1.	Unzip the files contained in the Shvan_VMP.rar file to a folder on your hard drive.  
2.	Open this project in your Netbeans IDE.
3.	Run (Computaion.java) file.
4.	First, we need a choice second option to create a dataset.
5.	Enter the number of ports of switches (basically switches comes 4 ports, 8 ports, 16 ports, 24 ports, 32 ports And so on) only enter the number.
6.	Enter the number of applications depends on three different experiments. 
7.	Automatically created a data set file in the same directory with the name of the number of port switches and application number that already entered.
8.	Then rerun the project and choose the first option (Load dataset) to get the result of 4 different algorithms. 
9.	Finally, we can repeat this process to create a new dataset and get the results.
