Step of the running project:

1. Download Matlab 2016B; Processor:  Intel(R) Core(TM) i7-8565U CPU @ 1.80GHz   1.99 GHz, Windows edition: Windows 10 Home Edition   
2. Using the CEC 2017 function set;
3. Download the program provided in this article;
4. Run the main.m program (our code); Drawing program is plot_fig.m; compariosn codes are OED.m and R_OED.m
5. Get the average value, best value, worst value, variance and so on according to the files generated by the program running (can check the DHEM folder)
