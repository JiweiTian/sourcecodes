Step of the running project:

1. Download Eclipse or Myeclipse Toolkit. We use Eclipse SDK-Version: Mars (4.5) (it can work on any Eclipse version 2020+);
2. Importing files such as " BFS.class", "Edge.class ", " FileReader.class", "Flow.class", " Graph.class" and  "Main.class " ;
3. Open the  "Main.class " file;
4. Run the Main.class file and input the number of nodes and each node names;
5. Obtain the experimental results.
